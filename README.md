# Sales-Customer-Analytics-Dashboard-Power-BI


This project is a multi-page interactive Power BI dashboard delivering comprehensive insights into sales, returns, and customer behavior.
It covers Sales Overview, Customer History, Customer Preferences, and Department & Material analysis with FY-based time intelligence.
Key features include dynamic slicers, KPI cards, YoY growth analysis, return ratio monitoring, and process/plant-wise performance views.
Advanced DAX calculations and conditional formatting highlight trends, risks, and variances.
The dashboard helps business teams track performance, control returns, and optimize customer and material strategies.
